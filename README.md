#### 步骤：
##### 项目创建

1. 新建项目文件夹 $ mkdir project
1. 进入项目文件夹 $ cd project
1. 项目仓库创建 $ git init
1. 生成.gitignore文件 $ touch .gitignore
1. 生成README.md文件 $ touch README.md
1. 安装cnpm $ cnpm install
1. 生成package.json文件 $ cnpm init
